# FormBuilder PWA

## How to use locally
1. Download all files from this folder
2. Put them in the same directory  
3. Open index.html in your browser
4. IMPORTANT: In index.html, find line ~295 and replace YOUR_GITHUB_TOKEN_HERE with the token from config.js

## Files
- index.html - Main PWA app
- manifest.json - PWA manifest  
- sw.js - Service worker
- config.js - Contains your GitHub token (split in two parts)
